# g1_robot_sim

## Introduction

This package simplifies working with IsaacSIM.

## Install

### Prerequesets

1. Install isaacsim, and isaaclab as [described here](https://isaac-sim.github.io/IsaacLab/main/source/setup/installation/pip_installation.html)
2. Install this package `pip3 install -e .` inside the isaacsim venv.

### Run Robot G1 simulation

Run `python apps/g1_simple.py` to open Robot g1 with articulation controller for moveit2.
